# study-tracker-app
-----------------------
A mobile app project to help students plan, track, and manage academic tasks and deadlines effectively.

-----------------------
## Overview
-----------------------
This project is a mobile application designed to help students plan, track, and manage their academic tasks and deadlines effectively.

-----------------------
## Motivation
-----------------------
As a high school student, I noticed that many students struggle with organizing subjects, assignments, and exam preparation. I decided to build this app to create a simple and structured way for students to manage their studies and reduce academic stress.

-----------------------
## Current Status
-----------------------
- Project initiated: 06-01-2026
- Planning and documentation phase
- UI and development to begin soon

-----------------------
## Planned Features (Initial Version)
-----------------------
- Add subjects
- Create tasks with deadlines
- Mark tasks as completed
- View daily and weekly task lists
